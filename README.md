# BEMSIChat
***
BEMSI project

### The aim of the project:
The aim of the project is to create end-to-end (OTR) chat.

### Technology stack:
- Java 11
- Kotlin
- SpringBoot
- React
- Gradle
- Git

### Build instructions

Gradle is required to the run this project. It's available [here](https://gradle.org/install/).

```
./gradlew web:bootRun
```
